# A Brainfuck Interpreter Made With C++

## 1. How To Use :
```bash
./{program name} {brainfuck file}
```


## 2. How To Compile :
### 1. You need to make the build directory
```bash
mkdir build
cd build
```

### 2. Generate build system with ninja
```bash
cmake -G Ninja ..
```

### 3. Compile with ninja
```bash
ninja
```
